##  Project Overview
This project is developed as part of **Assignment 1** for the Web Development course.  
The goal is to design a **responsive, mobile-first landing page** themed around **SDG 13: Climate Action**,  
focusing on *Urban Heat & Flood Resilience*.  

The page aims to promote awareness about environmental challenges,  
showcase event details for the Climate Action Hackathon 2025,  
and encourage user participation through a clean, accessible web interface.

##  Technical Features
-  **Semantic HTML5** tags used for structure (header, main, section, footer).  
-  **CSS Flexbox and Grid** for layout alignment and spacing.  
-  **Responsive (mobile-first)** design using media queries at 768px and 1024px breakpoints.  
-  **Accessibility compliance** with labeled form fields, high contrast colors, and skip link navigation.  
-  **Clean navigation bar** that adjusts layout for smaller screens.  
-  **Consistent color palette** inspired by nature and sustainability (green tones).  
-  **Optimized for readability** using simple typography and balanced spacing.  
-  **No external libraries** — built purely with HTML and CSS.

---

##  Responsive Design (Mobile-First)
The page layout is fully responsive:  
- On mobile: single-column layout, stacked sections, readable text.  
- On tablet: medium grid layout for cards and forms.  
- On desktop: multi-column layout for speakers and sponsors.  

Media queries ensure that all elements adjust smoothly without overflow or distortion.

The design uses green shades to represent sustainability and growth:  
| Color | Code | Meaning |

| Light Green Accent | `#a5d6a7` | Growth, renewal |

| Text Color | `#222` | Neutral and readable |

Typography: **Arial / Sans-serif**, ensuring accessibility and clarity.  
Overall visual tone: **minimal, clean, and professional.**
